# residencia-tecnologica-globo-libras-ai
Projeto desenvolvido na Residência Tecnológica do Porto Digital em parceria com a Globo, com o objetivo de automatizar a geração de tradução em Libras para vídeos. O sistema é baseado em um pipeline que inclui processamento de áudio, transcrição, tradução para Libras e renderização com avatar, permitindo acessibilidade em escala.
